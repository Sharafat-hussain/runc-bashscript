# runc-bashscript

For Linux
Bashscript to compile and run your C code.

## Installation

Copy **runc** file in your code directory.

## Usage

Copy runc file in your code directory. To run foo.c enter this to Terminal
```bash
./runc foo.c 
```
OR
```bash
bash runc foo.c
```
If file name is main.c only enter

```bash
./runc
```
OR
```bash
bash runc
